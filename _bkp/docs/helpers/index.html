Helpers

This is a list on all the helpers for templates and styles and how to use them

Template helpers

These are simple helpers to speed up layout development.

⚠️ They are imported on /src/website/global.pug and should only be imported there, and only once.

Location: /src/./_helpers.pug

+stylesheet(options:Object)

Use this to create <link> tags to local and external stylesheets.

⚠️ ONLY used under the styles block.

Options

filename: String

required

The file name, you should omit the extension and path, just use the filename.

isExternal: Bool

optional

If the file is from an external source, set this to true

Usage

append styles
  +stylesheet({
    filename: 'home'
  })

+javascripts(options:Object)

Use this to create <script> tags to local and external javscripts.

⚠️ ONLY used under the javascripts block.

Options

filename: String

required

The file name, you should omit the extension and path, just use the filename.

isExternal: Bool

optional

If the file is from an external source, set this to true

isLib: Bool

optional

If the file is an library but not from an external source

⚠️ Library files should be added to /src/assets/vendor/

Usage

append javascripts
  +script({ filename: 'three.min', isLib: true })

+img(options:Object)

Use this to create <img> tags to local assets.

⚠️ ONLY used for local assets.

Options

src: String

required

⚠️ Use the srcset option if you have the -2x version of the image.

The path to the image inside /src/website/assets/img/.

alt: String

optional

If no alt is passed, an empty alt will be used for accessibility purposes.

srcset: String

optional

Same as src option but the <img> tag generated will have a srcset attribute set to the -2x version of the image.

⚠️ The -2x version of the file should have the same name as the 1x version and be suffixed with -2x

⚠️ Don't use src and srcset together, just use one. If you don't have a -2x version of an image, just use src.

Usage

+img({ srcset: 'home/main.png', alt: 'Sweet potato' })
