Reduce Motion

A collection of scripts and classes to help you create a more acessible website.

This component is composed of 4 files, all located on the same folder. We will go through each file and what they do.

Location: /src/components/reduce-motion/

Markup

This button can be included anywhere on the website, it toggles the reduce motion flag on and off.

✅ By default it's included in the main-nav.pug.

Location: /src/components/reduce-motion/reduce-motion.pug

Usage

include ./reduce-motion/reduce-motion.pug

Styles

The main reduce-motion.scss file includes the basic styles for the button to toggle the icons and the label.

✅ By default it's included in the global.scss.

Location: /src/components/reduce-motion/reduce-motion.scss

Styles Helper

The helper _reduce-motion.scss file includes a mixin to be used when creating a reduce motion version of a component.

Location: /src/components/reduce-motion/_reduce-motion.scss

@include reduce-motion

Usage

@import './include/reduce-motion/reduce-motion';

.component {
  transition: transform 0.2s ease, opacity 0.2s ease;

  @include reduce-motion {
    transition-property: transform;
  }
}

Scripts

The scripts help when you need to dynamically toggle the reduce motion on and off.

Location: /src/components/reduce-motion/reduce-motion.js

Usage

import { rmGetState, rmOn, rmOff, rmInit, rmDestroy } from "../../../components/reduce-motion/reduce-motion.js";

// Initiate it first
rmInit();

// after you initiate you can start listening to events
window.addEventlistener('wtc-reduce-motion', function({ detail }) {
  const { state } = detail;
  if (state) {
    // reduce motion is on
  }
  else {
    // reduce motion is off
  }
});

// Turn reduce motion on
rmOn();

// Turn reduce motion off
rmOff(); 

// Get current state
const isReduceMotion = rmGetState();

// this is used to remove the click event on the buttons
// you will use this for ajax websites
rmDestroy();
